Installing R and RStudio

For this training, we will use:

  1. R ‚Äì a free software environment for statistical computing and graphics (download and install from https://cloud.r-project.org/)
  2. RStudio ‚Äì an integrated development environment for R (download and install from https://rstudio.com/products/rstudio/download/)
  3. Packages ‚Äì extend the capabilities of R. To install a package, open RStudio then
    • Packages ‚Äì> Install ‚Äì> type the package name in Packages ‚Äì> Install or
    • Type install.packages("package_name") then hit Enter key
      • e.g.: install.packages(“plotly”)
